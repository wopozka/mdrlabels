# mdrlabels
Creataing labels for medical devices. Combanining static label elements (symbols) with dynamic (use by date, batch, UDI-PI).

## Kluczowe koncepty

### App_ID (Application Identifier)
Każde pole w systemie identyfikowane jest poprzez **App_ID** - unikalny identyfikator zgodnie ze standardami zarządzania identyfikatorami aplikacji. App_ID jest używany do:
- Mapowania danych z formularza na pozycje w dokumencie PDF
- Przechowywania wartości pól w bazie danych
- Komunikacji między komponentami aplikacji
- Zapewnienia spójności i unikania konfliktów nazewnictwa
